Welcome to the visual guide of sorting algorithms. Understanding how computers organize data is fundamental to computer science.

1. Bubble Sort

Bubble Sort is the simplest sorting algorithm that works by repeatedly swapping the adjacent elements if they are in the wrong order.

Steps:

  1. Compare adjacent elements.
  2. Swap if they are in wrong order.
  3. Repeat until sorted.

2. Merge Sort

Merge Sort is a "Divide and Conquer" algorithm. It divides the input array into two halves, calls itself for the two halves, and then merges the two sorted halves.

3. Quick Sort

QuickSort is also a Divide and Conquer algorithm. It picks an element as a pivot and partitions the given array around the picked pivot.
