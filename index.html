Introduction to Sorting Algorithms

A visual guide to understanding how computers organize data.

1. Bubble Sort

Bubble Sort is the simplest sorting algorithm that works by repeatedly swapping the adjacent elements if they are in the wrong order.

Key Characteristics:

  • Complexity: O(nÂ²) - Very slow on large lists.
  • Method: Exchanging (Swapping).
  • Best for: Educational purposes and very small datasets.

2. Selection Sort

The selection sort algorithm sorts an array by repeatedly finding the minimum element from the unsorted part and putting it at the beginning.

Steps to perform:

  1. Set the first element as the minimum.
  2. Compare minimum with the second element. If the second element is smaller, assign it as the new minimum.
  3. After iterating through the list, swap the minimum with the first element.
  4. Repeat for the rest of the list.

3. Insertion Sort

Insertion sort works similarly to the way you sort playing cards in your hands. The array is virtually split into a sorted and an unsorted part.

4. Merge Sort

Merge Sort is a "Divide and Conquer" algorithm. It divides the input array into two halves, calls itself for the two halves, and then merges the two sorted halves.

Why use it?

  • Complexity: O(n log n).
  • Stability: It preserves the order of equal elements.
  • Usage: Great for sorting linked lists.

5. Quick Sort

QuickSort is also a Divide and Conquer algorithm. It picks an element as a pivot and partitions the given array around the picked pivot.

How it partitions:

  1. Choose a pivot (e.g., last element).
  2. Move all smaller elements to the left of the pivot.
  3. Move all greater elements to the right of the pivot.
  4. Recursively apply to sub-arrays.
